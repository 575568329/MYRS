## 🚀 快速入门

### 第一次使用

#### 步骤 1: 安装插件

**方式 A: 从 uTools 插件市场安装（推荐）**
1. 打开 uTools
2. 点击左侧插件图标
3. 在搜索框输入"摸鱼热搜"
4. 点击"安装"按钮

**方式 B: 本地安装**
1. 下载项目源码
2. 在 uTools 中进入 `设置` → `开发者模式`
3. 点击 `安装本地插件`
4. 选择项目文件夹
5. 点击"安装"

#### 步骤 2: 唤醒插件

在 uTools 中输入以下任意关键词：
- `热搜`
- `热榜`
- `hot`
- `摸鱼`

插件窗口会立即弹出，显示默认平台（百度）的热搜列表。

#### 步骤 3: 浏览热搜

1. **切换平台** - 点击顶部的平台标签
2. **查看详情** - 点击任意热搜条目
3. **刷新数据** - 点击右下角 🔄 按钮
4. **打开设置** - 点击右下角 ⚙️ 按钮

### 顶部区域

#### 1. 平台分类（仅分类模式）
- **位置**: 最顶部
- **功能**: 切换不同分类（综合、社交、视频等）
- **样式**: 圆角按钮，选中时蓝色高亮

**可用分类**:
- 📊 综合
- 💬 社交
- 🎬 视频
- 📰 资讯
- 💻 科技
- 🎮 娱乐
- 🎯 游戏
- 📚 阅读

#### 2. 当前平台显示
- **位置**: 平台标签左侧
- **样式**: 蓝色渐变背景，独立于标签列表
- **显示内容**: 平台图标 + 平台名称
- **特点**: 始终固定在左侧，不随标签滚动

#### 3. 平台标签
- **位置**: 当前平台右侧
- **功能**: 切换不同平台
- **交互**:
  - 点击切换平台
  - 支持横向滚动（鼠标滚轮）
  - 左右箭头提示可滚动

### 内容区域

#### 热搜列表项

每条热搜包含：

| 元素 | 说明 | 样式 |
|------|------|------|
| **排名** | 圆形数字徽章 | 前三名橙色，其他灰色 |
| **标题** | 热搜标题 | 蓝色，可点击 |
| **简介** | 热搜描述（可选） | 灰色小字 |
| **热度** | 热度数值 | 橙色，右侧显示 |

#### 排名样式

```
1  → 橙色圆点 (🔥)
2  → 橙色圆点 (🔥)
3  → 橙色圆点 (🔥)
4  → 灰色圆点
5  → 灰色圆点
...
```

#### 热度显示

支持两种模式（可在设置中切换）：

1. **数字模式**: 🔥 1234万
2. **等级模式**: 🔥🔥🔥🔥🔥 (爆)

热度等级划分：
- 🔥🔥🔥🔥🔥 爆 - 1000万+
- 🔥🔥🔥🔥 热 - 500万+
- 🔥🔥🔥 温 - 100万+
- 🔥🔥 凉 - 50万+
- 🔥 冷 - 50万以下

### 悬浮按钮

#### 位置
- 固定在右下角
- 距离底部 24px，距离右侧 24px
- 不随页面滚动

#### 刷新按钮（🔄）
- **颜色**: 蓝色渐变
- **功能**: 刷新当前平台数据
- **状态**:
  - 默认: 蓝色，可点击
  - 加载中: 旋转动画，禁用
  - 悬停: 上移 2px + 放大 5%

#### 设置按钮（⚙️）
- **颜色**: 灰色渐变
- **功能**: 打开设置面板
- **状态**:
  - 默认: 灰色，可点击
  - 悬停: 上移 2px + 放大 5%

## 🔧 功能详解

### 1. 切换平台

#### 方法 1: 点击平台标签
- 直接点击任意平台标签
- 该平台的热搜会立即加载

#### 方法 2: 使用平台分类（分类模式）
1. 点击顶部分类标签
2. 在下方选择具体平台
3. 分类会自动保存，下次打开生效

#### 自动记忆
- 插件会自动记住你最后使用的平台
- 下次打开时自动加载该平台
- 分类选择也会被记住

### 2. 查看热搜详情

#### 在 uTools 中
- 点击热搜条目
- 链接会在**默认浏览器**中打开

#### 外部浏览器
- 支持所有主流浏览器
- 自动调用系统默认浏览器

### 3. 刷新数据

#### 手动刷新
1. 点击右下角 🔄 按钮
2. 按钮开始旋转（加载中）
3. 数据更新完成后停止旋转

#### 自动刷新时机
- 切换平台时
- 第一次打开插件时
- 手动点击刷新按钮时

#### 数据更新频率
- API 数据源缓存: ~60分钟
- 建议每小时手动刷新一次
- 频繁刷新不会获取到新数据（缓存限制）

### 4. 分页加载

#### 自动加载
- 滚动到列表底部
- 距离底部 100px 时自动触发
- 显示"加载中..."动画

#### 手动加载
- 滚动到底部时显示"加载更多"按钮
- 点击按钮加载下一页

#### 加载状态
- **加载中**: 显示旋转动画
- **没有更多**: 显示"已经到底了"
- **加载失败**: 显示错误提示

#### 每页显示数量
- 默认: 50 条
- 可在 `config.js` 中配置 `PAGINATION.DEFAULT_PAGE_SIZE`

### 5. 平台标签滚动

#### 鼠标滚轮滚动
- 鼠标悬停在标签区域
- 使用滚轮上下滚动
- 标签会自动左右移动

#### 点击箭头滚动
- 左箭头 `‹`: 向左滚动 70% 宽度
- 右箭头 `›`: 向右滚动 70% 宽度
- 箭头仅在可滚动时显示

#### 滚动指示器
- 左侧渐变: 提示左侧有更多内容
- 右侧渐变: 提示右侧有更多内容
- 半透明白色渐变遮罩

## ⚙️ 个性化设置

### 打开设置面板

点击右下角 ⚙️ 按钮，打开设置面板。

### 主题模式

#### 自动模式（推荐）
- 跟随系统主题设置
- Windows: 设置 → 个性化 → 颜色 → 选择模式
- macOS: 系统偏好设置 → 通用 → 外观
- Linux: 根据桌面环境设置

**切换时间**:
- 实时监听系统主题变化
- 无需重启插件
- 自动应用新主题

#### 亮色模式
- 始终使用浅色主题
- 适合白天使用
- 节省电量（OLED 屏幕除外）

#### 暗色模式
- 始终使用深色主题
- 适合夜间使用
- 保护眼睛，减少蓝光

### 显示选项

#### 显示热度值
- **开启**: 在每条热搜右侧显示热度
- **关闭**: 隐藏热度信息
- **默认**: 开启

**显示格式**:
- 🔥 1234万
- 🔥 567.8万
- 🔥 12万

#### 显示简介
- **开启**: 在标题下方显示热搜简介
- **关闭**: 只显示标题
- **默认**: 开启

**影响**:
- 开启后列表项高度增加
- 关闭后列表更紧凑，可显示更多内容
- 如果某条热搜没有简介，则不显示

### 平台顺序

#### 自定义排序（极简模式）
1. 打开设置面板
2. 找到"平台顺序"部分
3. 拖拽调整平台顺序
4. 顺序自动保存到本地

#### 重置顺序
- 删除本地存储数据
- 恢复为配置文件中的默认顺序

#### 应用场景
- 将常用平台放在前面
- 根据使用频率排序
- 个人偏好定制

## 🔍 故障排除

### 问题 1: 平台加载失败

#### 可能原因
1. **API 服务器不可用**
   - 检查网络连接
   - 访问 [API 状态页](https://api-hot.imsyy.top)

2. **跨域限制**
   - 建议在 uTools 环境中使用
   - 浏览器中可能有跨域问题

3. **平台接口更新**
   - 部分平台接口可能失效
   - 等待 API 更新

#### 解决方案
```bash
# 1. 刷新页面
点击右下角 🔄 按钮

# 2. 切换平台
尝试加载其他平台

# 3. 检查网络
ping api-hot.imsyy.top

# 4. 查看控制台
按 F12 打开开发者工具，查看错误信息
```

### 问题 2: 主题切换无效

#### 可能原因
1. **浏览器缓存**
2. **设置未保存**
3. **本地存储被清空**

#### 解决方案
```bash
# 1. 清除缓存
uTools → 设置 → 清除缓存

# 2. 重新设置
打开设置面板，重新选择主题

# 3. 重启插件
关闭插件窗口，重新打开
```

### 问题 3: 平台顺序不保存

#### 可能原因
1. **本地存储被禁用**
2. **存储空间不足**
3. **数据格式错误**

#### 解决方案
```bash
# 1. 检查 uTools 存储权限
uTools → 设置 → 隐私 → 允许插件存储数据

# 2. 清空旧数据
uTools → 设置 → 开发者模式 → 清除数据

# 3. 重新排序
打开设置，重新拖拽排序
```

### 问题 4: 悬浮按钮不显示

#### 可能原因
1. **z-index 层级问题**
2. **CSS 加载失败**
3. **窗口尺寸过小**

#### 解决方案
```bash
# 1. 调整窗口大小
增大 uTools 窗口尺寸

# 2. 刷新插件
关闭并重新打开插件

# 3. 检查更新
确保使用最新版本
```

### 问题 5: 数据显示异常

#### 可能原因
1. **API 数据格式变化**
2. **解析逻辑错误**
3. **网络传输错误**

#### 解决方案
```bash
# 1. 刷新数据
点击 🔄 按钮重新加载

# 2. 切换平台
尝试其他平台是否正常

# 3. 查看控制台
F12 → Console 标签查看错误
```
